# Centro Espírita Falangeiros da Luz (CEFL) 🕊️

Este é o site oficial do **Centro Espírita Falangeiros da Luz**, uma casa de Umbanda localizada na Piedade, Rio de Janeiro. O projeto foi desenvolvido para fornecer informações sobre a casa, agenda de giras, regras de conduta, localização e contatos para a comunidade.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web modernas para garantir um layout responsivo e dinâmico:

* **HTML5:** Estrutura semântica do site.
* **CSS3:** Estilização modularizada (utilizando múltiplos arquivos para melhor organização).
* **JavaScript (JS):** Lógica de interatividade e inicialização de componentes.
* **Swiper.js:** Biblioteca externa utilizada para o carrossel (slider) de fotos na seção principal.
* **Font Awesome:** Ícones para as redes sociais e contatos.
* **Google Maps API:** Integração de mapa para localização da casa.

## 📁 Estrutura de Arquivos

O projeto está organizado da seguinte forma:

* `index.html`: Arquivo principal da página.
* `css/global/index/index.css`: Arquivo central que importa todos os módulos de estilo.
* `css/`: Pasta contendo arquivos CSS específicos para cada seção (`header.css`, `main.css`, `section-agenda.css`, etc.).
* `js/`: Contém o arquivo `index.js` para controle do Swiper e outras interações.
* `css/imagens/`: Armazena as fotos e o favicon do site.

## ⚙️ Funcionalidades

* **Carrossel Responsivo:** Galeria de fotos que se adapta a dispositivos móveis e desktops.
* **Agenda Dinâmica:** Seção detalhada com as datas e eventos do mês.
* **Menu de Informações (Accordion):** Seção de regras e informações utilizando a tag `<details>`, permitindo que o usuário expanda o conteúdo que deseja ler.
* **Botões de Contato Direto:** Links integrados com o WhatsApp para agendamento de consultas e dúvidas.
* **Design Dark Mode:** Estética moderna com fundo escuro e detalhes em dourado (`#d4af37`), remetendo à identidade visual da casa.
